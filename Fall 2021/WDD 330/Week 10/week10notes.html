Questions

I need to figure out a way for the note title on the side to update when the user types in the note, but I can't find anything on Google. Any advice?

  • Client side validation: ensure all required form controls are filled out, in the correct format
  • form validation: When you enter data, the browser and/or the web server will check to see that the data is in the correct format and within the constraints set by the application
  • You can use the built in validation, or create it yourself using javascript
  • Build in
    • The element needs to match the :valid or :invalid psudeo classes
    • To mark an item as required use :required
  • Regular expression: pattern that can be used to match character combinations in text strings
  • Validating with Javascript:
    • Constraint validation API: consists of a set of methods and properties available on the following form element DOM interfaces
    • You can also implement a custom error message
  • Fetch: provides an easy way to fetch resources across the network. Useful for working with APIs
  • When you fetch it retuns a promise that is resolved using the response object
  • Fetch can accept a second parameter
  • If the fetch is rejected it will respond with a TypeError
  • You can also supply your own request using the Request() constructor, and pass it in as a fetch argument
  • Headers(): Constructor that allows you to create your own headers object
  • Guard: affects which mutation operations are allowed on the headers object
  • Polyfill: recreate the functionality for non-supporting browsers
