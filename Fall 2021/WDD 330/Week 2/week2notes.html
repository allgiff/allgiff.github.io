Week 2 Notes

Practice Exercises
  1. Use comments to explain things. This helps future you and anyone who looks at the code.
  2. Some words (like break and await) cannot be used as variable names
  3. Primitive Data Types (anything that isn't a data type is an object)
    • String
    • Symbol
    • Number
    • Boolean
    • Undefined
    • Null
  4. You can use typeof operator to find out value types
  5. Const: use for constant variable. Only have value in block it is palced in.
  6. Let: for a unconstant variable. Only have value in block it is palced in.
  7. Global Scope: Any variable declared outside a block and can be used anywhere in program. NOT good practice
  8. Local scope: Variables declared in their block will not have value outside it
  9. Variable names should describe what it means
  10. Method: action a primitive data type or object can perform
  11. Template Literal: use backtick to deliminate string
  12. NaN: error value meaning 'not a number'
  13. Booleans: Can have a value of true or false
  14. There are only seven values that are false in JavaScript and these are known as 'falsy' values.
  15. Type coercion is when JavaScript tries to convert a value into another data type in order to perform an operation
  16. Values can be compared to see if they are equal, greater than or less than other values
  17. Array: ordered list of values
    • Heroes: empty array
    • Multidimensional: arrays that contain other arrays
    • Destructuring: taking values out of an array and presenting them as indivigual values
    • .length: find the length of an array
    • The switch statement can be used instead of multiple if and else statements
    • pop(): removes the last item of the array
    • shift(): removes the first item in an array
    • push(): appends a new value to the end of the array
    • unshift(): appends new item to array beginning
    • concat(): merges arrays
    • join(): Turns an array into a string
    • slice(): creates a subarray
    • reverse(): Reverses the order of the array
    • sort(): sorts an array, permanantly changing the array order
  18. Set: Data structure that represents a collection of unique values. No duplicate values
  19. size(): Finds the number of values inna set
  20. has(): Checks if a value is in a set
  21. You can convert a set into an array using spread operator
  22. While loop: Repeatedly runs a block of code while a certain condition is true
  23. HTML = Content
  24. CSS = Presentation
  25. JavaScript = Interactivity
  26. Graceful degradation: Process of building a wabsite so it works best in modern and old browsers.
  27. Progressive enhancement: process of building a page from the ground up with base level functionality, then adding enhancements.
  28. Transpiler: converts JavaScript code to different versions
