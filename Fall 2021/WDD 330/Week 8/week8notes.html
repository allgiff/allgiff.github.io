  • Transform: Property that lets you translate, rotate, scale and skew any element
    • Transform Functions: Used to manipulate an elements appearance
    • Translation: Allows you to move objects left, right, up, or down
    • Scaling: Allows you to grow or shrink and element using x,y values
    • Rotation: Rotates an element around a point of orgin (which can be changed)
    • Skew: Skews an element along x,y axes
  • Transitions: Allows css property values to change over time (basically forms simple animations)
    • Steps to create a simple transition:
      • 1. Declare the original state of the element in the default style declaration
      • 2. Declare the final state of your transitioned element; for example, a :hover state
      • 3. Include the transition functions in your default style declaration using the transition properties
    • Transition Property: defines css properties of the element that will be transitioned
    • Transition-Duration: Allows you to set how long the transition takes
    • Transition properties allow for multipul transitions in one call (ie. color and rotation
  • Animations: Created using @keyframes
    • Animations must be called using animation in the css for the item that is changing
    • Animation-iteration-count: Allows you to define how many times an animation plays
    • Animation-direction: changes the direction of an animation
    • make sure not to use words like forwards or running for animation names
